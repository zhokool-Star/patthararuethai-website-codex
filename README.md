# Cosmic Energy Site

เว็บไซต์หน้าเดียวสำหรับ **อาจารย์ภัทรฤทัย ปราญ์ชมุณีภร | พยากรณ์บ้านดวงดี** สร้างด้วย Vite, React และ Three.js

## Features

- หน้าเว็บภาษาไทยแบบ one-page
- ธีมพลังงานจักรวาล / เช็คพลังงานดวงดาว
- 3D animation ด้วย Three.js
- ส่วนเช็คพลังงานแบบ interactive
- Responsive สำหรับ desktop และ mobile

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
